View属性
  • SWITCHTYPE_HISTORYFORWARD - 常量：视图切换操作类型：由浏览器前进操作触发
  • SWITCHTYPE_HISTORYBACK - 常量：视图切换操作类型：由浏览器后退操作触发
  • SWITCHTYPE_VIEWNAV - 常量：视图切换操作类型：由视图切换：View.navTo操作触发
  • SWITCHTYPE_VIEWCHANGE - 常量：视图切换操作类型：由视图切换：View.changeTo操作触发
  • currentState - {JsonObject} 当前的历史状态（等同于history.state）
  • Logger - {Function} 日志输出组件构造器
  • layout - {JsonObject} 布局工具组件
