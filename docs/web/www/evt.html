View实例 - ready
视图就绪。该事件仅在对应的视图第一次进入时触发。触发前已完成活动视图的切换和视图布局动作的执行。
{sourceView：[源视图], type: [切换类型], params: [视图参数]}
同步
View.ofId("detail").on("ready", function(e){});
下一节
View实例 - beforeenter
视图进入前。如果视图尚未就绪，则在ready事件之前触发。触发时，活动视图的切换和视图布局动作尚未执行。
{sourceView：[源视图], type: [切换类型], params: [视图参数]}
同步
View.ofId("detail").on("beforeenter", function(e){});
下一节
View实例 - enter
视图进入时。如果视图尚未就绪，则在ready事件之后触发。
{sourceView：[源视图], type: [切换类型], params: [视图参数]}
同步
View.ofId("detail").on("enter", function(e){});
