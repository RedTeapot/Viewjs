View.navTo(targetViewId[, ops])
以“压入历史堆栈”的方式切换视图。
  1. targetViewId – {String} 目标视图ID，或伪视图，或视图群组
    支持的伪视图：
    • :back – 回退
    • :forward – 前进
    • :default-view 默认视图
    视图群组格式：“~groupName”，如：View.navTo(“~profile”)。
  2. ops – {JsonObject} 可选。切换动作的控制选项
    • ops.withAnimation – {Boolean} 是否播放动画。默认为：true
    • ops.params – {JsonObject} 视图参数集合。key：参数名称；value：参数取值。默认为：null
    • ops.options – {JsonObject} 视图选项集合。key：选项名称；value：选项取值。默认为：null
{Function} – View静态对象
