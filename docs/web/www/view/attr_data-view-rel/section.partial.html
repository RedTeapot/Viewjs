data-view-rel
data-view-rel
响应用户触摸，跳转到指定的目标位置。支持伪视图和视图群组。支持视图选项，如：<span data-view-rel=”category!id=1”>分类</span>
  • 任一存在的视图编号，亦即对应的DOM元素的ID；
  • 伪视图：":back"（后退），":forward"（前进），":default-view"（默认视图）
  • 视图群组
  • 其它链接

当配置为视图群组时，需要添加特殊符号：“~”，如：<span data-view-rel="~category">分类</span>。

当取值为视图编号，或视图群组时，开发者可以使用”!”作为分隔符传递参数。如：<span data-view-rel=”category!id=1”>分类</span> <span data-view-rel=”~category!id=1&name=XXXX”>分类</span>。多个参数之间使用符号：“&”分隔。

当配置为其它链接地址时，需要添加特殊符号：“@”，如：<span data-view-rel="@index.html">首页</span>
