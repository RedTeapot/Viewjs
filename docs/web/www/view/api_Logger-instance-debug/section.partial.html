logger.debug(template[, placeHolderValues...])
以DEBUG级别输出日志，如：
var logger = View.ofId(“detail”).logger; // -> 0629 09:34:51 [View#detail]: hello, world logger.debug("hello, {}", "world"); // -> 0629 09:34:51 [View#detail]: Dynamic contents: true, 12, text {} logger.error("Dynamic contents: {}, {}, {} \\{}", true, 12, 'text', 33, 45); logger.setIsEnabled(false); logger.info("No output would be generated");
  1. template – {String} 包含占位符的输出内容模板。占位符：{}，使用“\”进行转义。
  2. placeHolderValues – {Any} 分别对应于内容模板中占位符的取值。参数个数不限。
