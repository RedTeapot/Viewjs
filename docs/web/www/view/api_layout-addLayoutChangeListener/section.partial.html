View.layout.addLayoutChangeListener(listener)
添加“布局发生改变”事件监听器。
  1. listener – {Function} 监听器。方法可以接受4个参数：layoutWidth, layoutHeight, browserWidth, browserHeight，分别代表布局宽度、布局高度、浏览器宽度，浏览器高度。如：
    View.layout.addLayoutChangeListener(function(layoutWidth, layoutHeight, browserWidth, browserHeight){ //… });
{JsonObject} – View.layout对象
